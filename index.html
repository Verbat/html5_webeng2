Mit CSS formatierte Tabelle aller "XHTML 1.0 strict"-Elemente

In dieser Hausaufgabe sollen Sie auf Ihrer Website bei cloudControl eine XHTML-1.0-strict-Seite erstellen (ohne Bootstrap!). Die Seite soll eine "schön" formatierte Tabelle aller 77 Elemente zeigen mit den folgenden Spalten:

  1. Element-Tag mit den spitzen Klammern: <tag></tag> oder falls zulässig <tag />
  2. Elementbeschreibung
  3. Inline- oder Blockelement?
  4. Semantisches HTML?

CSS und HTML der Seite sind mit dem W3C-Validator zu validieren (keine Fehler, keine Warnungen).

Barrierefreies Suchformular

Sie sollen für die deutsche Wikipedia ein barrierefreies Webformular für die Suche nach Artikeln entwickeln. Als erste Orientierung kann Ihnen das Beispiel im Wikipedia-Artikel "Webformular" dienen:

<form action="http://de.wikipedia.org/wiki/Spezial:Suche">
<fieldset>
<legend>Suchen</legend>
<input type="text" name="search" value="" />
<input type="submit" name="go" value="Artikel"/>
<input type="submit" name="fulltext" value="Volltext" />
</fieldset>
</form>

Ihr Suchformular muss sich auf Ihrer Website in der Cloud von cloudControl befinden und den Test auf Einhaltung der Richtlinien WCAG 2.0 AAA und Section 508 bestehen.
